Hello! Tyria is a homebrew world that I've been using as a backdrop for heroic fantasy campaigns since ~2019! Over the years, I've gathered lore from my players, from the things I watch and read, and I've collated as much of it as I can within this very vault. Please, feel free to peruse, glance, or even skim these notes to your delight!
## Getting started
To the left is a search bar and a dropdown menu. If you know what document you're looking for -- great! You can just search directly for it. If you don't, you can try to find it using the dropdown links provided. 

You can also try searching for a document based on what campaign it was related to in (e.g. "\#BANE"). I try to tag notes with at least 1 campaign tag whenever possible, so hopefully that's a helpful line there. 

Lastly, you might try to find a document through related, linked documents, ala WikiLinks-style. You can hop from [[Ambrose Tenebris]] to [[Deimos Tenebris]] back to [[Ambrose Tenebris]], if you so wish.

If you'd like to see a map of the world, [here's my depiction of Tyria](https://imgur.com/a/MUABCHk), built using Inkarnate!
